## ⚡ What is this?

It's TicTacToe. But neon. But futuristic. But with a full terminal console, 35 achievements, AI difficulty modes, a Konami code easter egg, stat tracking, color customization, and a hidden developer mode you unlock by running fake npm commands.

So yeah. Just TicTacToe.

---

## 🎮 Features

### Core Gameplay
- **PvP** — Classic two-player local mode
- **Easy AI** — Moves randomly, you'll win, it's fine
- **Normal AI** — 50/50 between smart and dumb, keeps it spicy
- **Hard AI** — Full Minimax. Unbeatable. Don't even try.

### Visuals
- 🌌 Animated neon grid background
- ✨ Floating particle system (configurable)
- 🖱️ Custom crosshair cursor with **fading trail**
- 🎨 Full **RGB color picker** — customize literally everything
- 📱 Fully responsive — works on mobile too

### Stats & Persistence
- Per-player win/loss/draw tracking
- AI stats broken down by difficulty mode
- Win streaks + best streak tracking
- Export stats as `.txt` · Import them back
- Stats survive page reloads via localStorage

### 🏆 Achievements (35 total)
From *"First Blood"* (win your first game) all the way to *"Final Form"* (play 1250 games). Includes hidden achievements that only reveal themselves when triggered. No spoilers here.

> 💀 There's one called **Dragon's Bane**. You'll know when you find it.

### 💻 Built-in Terminal Console
Press **`~`** to open the command terminal. Available commands:

```
help                          show all commands
stats                         global stats overview
stats ai                      AI record by difficulty
reset all                     nuke everything
toggle particles              on/off
toggle konami                 you know what this does
export                        download your stats
set name <1|2> <name>         rename players
achievements all              list every achievement
devmode                       ...
```

### 🕹️ Konami Code
```
↑ ↑ ↓ ↓ ← → ← → B A
```
Activates binary rain background + full neon green mode. There's an achievement for it.

### 🔧 Developer Mode
There's a hidden dev mode that unlocks stat manipulation. Find it yourself — or read the code. The terminal will guide you if you know what to ask it.

## 🚀 Setup

It's a single HTML file. You literally just open it.

```bash
git clone https://github.com/The-Lazy-Dragon/lazy-dragons-tictactoe.git
cd lazy-dragons-tictactoe

# open tictactoe.html in your browser
# that's it
# seriously
```

No npm install. No build step. No dependencies. No node_modules folder eating your SSD. Just one `.html` file and a dream.
